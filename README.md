# DetectorPhishing
Proyecto académico que detecta phishing analizando únicamente la URL, sin contenido de la página. Incluye un modelo supervisado (Random Forest) y un análisis no supervisado (t-SNE + DBSCAN/K-Means) para entender patrones y outliers en los enlaces.
